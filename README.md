GPMC-York
=========
#####Java swing application to help teachers or adults manage stories and assign stories to different children.####

###Basic function:###

> Adult
- login/register
- manage stories(add, delete, edit, read)
- assign children
- preview story

<br>
> Children
- read stories
- make feedbacks
- login

###Extra function:###

- PDF printer
- spell checker
- voice reader
- page sound
- scroll page
